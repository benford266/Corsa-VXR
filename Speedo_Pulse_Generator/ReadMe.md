# GPS Pulse Generator
Pulse generator to emulate VSS sensor into EMU Black, Using GPS

## Wiring for ProMicro
 VCC- Plug Pin 1 / VCC GPS
 
 GND- Plug Pin 2 / GND GPS
 
 10- VSS Wire Plug Pin 3
 
 9- GPS RX
 
 8- GPS TX

## GPS
TX-Micro 8

RX-Micro 9

GND- Micro GND

VCC- Micro VCC

## Plug 
1 - Micro VCC / GPS VCC

2 - Micro GND / GPS GND

3 - Micro 10
